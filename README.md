
# Ecoleta-Frontend
Front-end made with React.js at NextLevel Week by rocketseat. Using geolocation and sending an image to the server

# Technologies used:
- React.js
- Dropzone
- Leaflet
- React Icons
- TypeScript

# New learnings
- FormData javaScript
- Upload images with Dropzone
- Geolocation with Leaflet
- React with typeScript
